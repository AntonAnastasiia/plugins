Posts from a Specific Category

* License: GPLv2 or later
* Requires PHP: 7.2
* Plugin URL: 

Adds a widget that shows the latest posts from one or more categories.

== Description ==
The Posts from a Specific Category widget is a lightweight widget designed to do one single job: display the latest posts from one or more specific categories.

== Features ==
* Ability to hide posts that do not have thumbnails.
* Option to show date of publication.
* Option to show thumbnail of publication.
* Option to show content of publication.

== Installation ==
= Automatic installation =
Automatic installation is the easiest option because WordPress handles the file transfer itself and you don't have to leave your web browser. To perform automatic installation of the Posts from a Specific Category,

1. Login to your WordPress dashboard, go to the Plugins menu and click Add New.
2. In the search field, type "Posts from a Specific Category" and click "Search Plugins".
3. Once you find the plugin, you can install it by simply clicking Install Now.
4. Then go to the WordPress admin plugins page and activate the plugin.
5. Now go to the Widgets page under the Appearance section and customize the Posts from a Specific Category widget.

= Manual installation =
1. Download the plugin.
2. Upload it to your blog's plugins folder.
3. Activate the plugin through the "Plugins" menu in WordPress
4. Now go to the Widgets page under the Appearance section and customize the Posts from a Specific Category widget.
